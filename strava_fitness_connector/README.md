# Strava Fitness MCP Add-on

Add-on per Home Assistant che espone API per:
- Sync attività Strava
- Metriche fitness
- Insight automatici

## Accesso

Apri l'add-on da Home Assistant per accedere a:
- /health
- /docs

## Configurazione

Inserisci:
- Strava Client ID
- Strava Client Secret

## Stato

Versione iniziale (scaffold)
